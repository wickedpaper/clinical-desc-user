Node.js Clinical DESC API

GET /desc to see all the policies.

GET /desc/{id} retrieves a desc user from the DB using an ID.

GET /db/{option} Create (and populate) or delete the DB. Option is either 'create' or 'delete.'

GET /fib Computes the Fibonacci sequence of 20.

PUT /desc/{id} updates a desc key using an ID.

DELETE /desc/{id} removes a policy from the DB using an ID.

POST /desc creates a new key.
  Ex: {name:'Basic',desc:'A low-cost package for the penny-pincher', baseCost:300,perAddTraveler:20,minDays: 0,perAddDay: 5,levelCare: 2,amount: 3000}
